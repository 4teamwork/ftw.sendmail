Introduction
============

With ``ftw.sendmail`` you are able to dynamically add tooltips to every DOM
element, which is selectable by jQuery and allows the title attribute.


Installing
==========

Add ``ftw.sendmail`` to your buildout configuration:

::

  [instance]
  eggs =
    ftw.sendmail
